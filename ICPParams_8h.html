Detailed Description

ICP algorithm parameters and configuration structures.

Author
Jens Munk Hansen
Date
2024-2025

This header defines a three-level parameter hierarchy for configuring the Iterative Closest Point (ICP) algorithm:

  • SessionParams: Session-level configuration including backend selection (hand-rolled vs Ceres), initial pose handling, and CUDA options.
  • OuterParams: Outer loop parameters controlling correspondence updates, subsampling rates, ray direction, and geometry-based weighting.
  • InnerParams: Inner loop solver parameters including solver type (Gauss-Newton or Levenberg-Marquardt), iteration limits, convergence thresholds, and Jacobian computation policy.

The Defaults namespace provides pre-configured parameter sets for common use cases: SinglePose, TwoPose, MultiView, and MultiViewCuda.
