Detailed Description

Left-perturbation ICP solver using tangent (6D) Jacobians.

This is a validation-only solver that uses left multiplication (space frame): T_new = Exp(delta) * T

Compare to ICPSolver.h which uses right multiplication (body frame): T_new = T * Exp(delta)

The two approaches should converge to the same solution.
