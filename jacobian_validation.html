Validating Left vs Right Jacobians on SO(3)

This document describes how to numerically validate Jacobians on SO(3) when using left (space / fixed frame) and right (body / moving frame) perturbations.

Problem Setup

Let \(R\) be a rotation matrix in \(\mathrm{SO}(3)\). Let \(f(R)\) be a vector-valued function.

Two common perturbation models:

Left perturbation (space frame):

\[ R(\varepsilon) = \exp(\varepsilon \, \widehat{\boldsymbol{\delta}}) \, R \]

Right perturbation (body frame):

\[ R(\varepsilon) = R \, \exp(\varepsilon \, \widehat{\boldsymbol{\delta}}) \]

The Jacobians depend on which perturbation model is used.

Finite-Difference Validation (Primary Test)

For a small \(\varepsilon\) and direction \(\boldsymbol{\delta} \in \mathbb{R}^3\), define the central difference:

\[ \frac{\mathrm{d}f}{\mathrm{d}\varepsilon}\bigg|_{\text{num}} = \frac{f(R_+) - f(R_-)}{2\varepsilon} \]

Use central differences for accuracy.

Left Jacobian Test

Perturbations:

\[ R_+ = \exp(\varepsilon \, \widehat{\boldsymbol{\delta}}) \, R, \qquad R_- = \exp(-\varepsilon \, \widehat{\boldsymbol{\delta}}) \, R \]

Analytical prediction:

\[ \frac{\mathrm{d}f}{\mathrm{d}\varepsilon}\bigg|_{\text{ana}} = J_{\text{left}}(R) \, \boldsymbol{\delta} \]

Validation criterion:

\[ \frac{\left\| \frac{\mathrm{d}f}{\mathrm{d}\varepsilon}\big|_{\text{num}} - \frac{\mathrm{d}f}{\mathrm{d}\varepsilon}\big|_{\text{ana}} \right\|} {\max\left(1, \left\| \frac{\mathrm{d}f}{\mathrm{d}\varepsilon}\big|_{\text{num}} \right\|\right)} \ll 1 \]

Right Jacobian Test

Perturbations:

\[ R_+ = R \, \exp(\varepsilon \, \widehat{\boldsymbol{\delta}}), \qquad R_- = R \, \exp(-\varepsilon \, \widehat{\boldsymbol{\delta}}) \]

Analytical prediction:

\[ \frac{\mathrm{d}f}{\mathrm{d}\varepsilon}\bigg|_{\text{ana}} = J_{\text{right}}(R) \, \boldsymbol{\delta} \]

Practical Notes

  • Use \(\varepsilon \sim 10^{-6}\) to \(10^{-8}\) (double precision)
  • Sweep \(\varepsilon\) over a decade and look for a plateau
  • Test random \(R\) and random \(\boldsymbol{\delta}\) (unit and scaled)
  • Compare directional derivatives, not full matrices

This catches sign errors, frame mixups, missing transposes, etc.

Cross-Validation Between Left and Right Jacobians

Left and right perturbations represent the same physical motion if their tangent vectors satisfy:

\[ \boldsymbol{\delta}_{\text{space}} = R \, \boldsymbol{\delta}_{\text{body}} \]

This implies the Jacobians must satisfy:

\[ J_{\text{right}}(R) = J_{\text{left}}(R) \, R \]

Equivalently:

\[ J_{\text{left}}(R) = J_{\text{right}}(R) \, R^\top \]

Numeric check:

\[ \| J_{\text{right}} - J_{\text{left}} \, R \| \approx 0 \]

If this fails but finite differences pass, your tangent convention differs (e.g., reversed delta mapping, quaternion local parameterization, etc.). An adjoint relationship still exists for the correct convention.

Simple Sanity Test Function

Use a function with a known derivative:

\[ f(R) = R \, \mathbf{p} \]

where \(\mathbf{p}\) is a fixed vector in \(\mathbb{R}^3\).

Left perturbation result:

\[ \mathrm{d}f = \boldsymbol{\delta}_{\text{space}} \times (R \, \mathbf{p}) \]

Right perturbation result:

\[ \mathrm{d}f = R \, (\boldsymbol{\delta}_{\text{body}} \times \mathbf{p}) \]

This test is extremely useful for debugging sign and frame errors.

Validation Checklist

  1. Finite-difference test \(J_{\text{left}}\)
  2. Finite-difference test \(J_{\text{right}}\)
  3. Verify \(J_{\text{right}} = J_{\text{left}} \, R\)
  4. Sanity check with \(f(R) = R \, \mathbf{p}\)

If all four pass, the Jacobians are correct.
