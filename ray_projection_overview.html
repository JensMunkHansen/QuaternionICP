Ray-Projection ICP Documentation

This page provides an overview of the point-to-plane ICP implementation using ray-projection residuals, with links to detailed documentation.

Introduction

Point-to-plane ICP minimizes the distance from source points to target surfaces along the surface normal direction. The ray-projection variant uses the residual:

\[ r = \frac{\mathbf{n}^\top(\mathbf{x} - \mathbf{q})}{\mathbf{n}^\top\mathbf{d}} \]

where:

  • \(\mathbf{x}\) is the transformed source point
  • \(\mathbf{q}\) is the target hit point
  • \(\mathbf{n}\) is the target surface normal
  • \(\mathbf{d}\) is the ray direction

This formulation requires careful handling of the Jacobians due to the quotient structure and the SE(3) manifold.
