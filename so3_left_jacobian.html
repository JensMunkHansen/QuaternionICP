SO(3) Left Jacobian

The SO(3) left Jacobian \(V(\boldsymbol{\omega})\) arises in the SE(3) exponential map, coupling rotation and translation. This document derives its closed-form expression and discusses numerical considerations.

Naming Convention

The term "left Jacobian" refers to \(V\) appearing on the left of \(\mathbf{v}\) in the translation formula \(V\mathbf{v}\), not to left-multiplication. This document uses the right-multiplicative (body-frame) SE(3) convention consistent with the implementation in SE3.h.

Context: SE(3) Exponential Map

The SE(3) exponential map for a twist \(\boldsymbol{\xi} = (\mathbf{v}, \boldsymbol{\omega})\):

\[ \exp(\widehat{\boldsymbol{\xi}}) = \begin{bmatrix} \exp([\boldsymbol{\omega}]_\times) & V(\boldsymbol{\omega})\,\mathbf{v} \\ \mathbf{0}^\top & 1 \end{bmatrix} \]

The translation component is \(V(\boldsymbol{\omega})\,\mathbf{v}\), not simply \(\mathbf{v}\). The matrix \(V\) accounts for the coupling between rotation and translation during the exponential map.

Definition

The SO(3) left Jacobian is:

\[ V(\boldsymbol{\omega}) = \sum_{k=0}^{\infty} \frac{1}{(k+1)!} [\boldsymbol{\omega}]_\times^k \]

This infinite series has a closed-form expression.

Closed-Form Expression

Let \(\theta = \|\boldsymbol{\omega}\|\) and define:

\[ [\boldsymbol{\omega}]_\times = \begin{bmatrix} 0 & -\omega_z & \omega_y \\ \omega_z & 0 & -\omega_x \\ -\omega_y & \omega_x & 0 \end{bmatrix} \]

Then:

\[ V(\boldsymbol{\omega}) = I + \frac{1 - \cos\theta}{\theta^2} [\boldsymbol{\omega}]_\times + \frac{\theta - \sin\theta}{\theta^3} [\boldsymbol{\omega}]_\times^2 \]

Using the scalar coefficients:

\[ B = \frac{1 - \cos\theta}{\theta^2}, \qquad C = \frac{\theta - \sin\theta}{\theta^3} \]

we write:

\[ V = I + B\,[\boldsymbol{\omega}]_\times + C\,[\boldsymbol{\omega}]_\times^2 \]

Special Cases

At Zero Rotation

\[ V(\mathbf{0}) = I \]

The translation passes through unchanged.

Small Angle Approximation

For \(\|\boldsymbol{\omega}\| \ll 1\):

\[ V(\boldsymbol{\omega}) \approx I + \frac{1}{2}[\boldsymbol{\omega}]_\times \]

This first-order approximation is used when \(\theta < 10^{-12}\) to avoid numerical instability in the trigonometric coefficients.

Derivation

From the Matrix Exponential

The SE(3) exponential can be written as:

\[ \exp\left(\begin{bmatrix} [\boldsymbol{\omega}]_\times & \mathbf{v} \\ \mathbf{0}^\top & 0 \end{bmatrix}\right) = \sum_{k=0}^{\infty} \frac{1}{k!} \begin{bmatrix} [\boldsymbol{\omega}]_\times & \mathbf{v} \\ \mathbf{0}^\top & 0 \end{bmatrix}^k \]

The translation block extracts as:

\[ V\,\mathbf{v} = \sum_{k=0}^{\infty} \frac{1}{(k+1)!} [\boldsymbol{\omega}]_\times^k \mathbf{v} \]

Using Rodrigues' Formula

From the SO(3) exponential (Rodrigues):

\[ \exp([\boldsymbol{\omega}]_\times) = I + \frac{\sin\theta}{\theta}[\boldsymbol{\omega}]_\times + \frac{1 - \cos\theta}{\theta^2}[\boldsymbol{\omega}]_\times^2 \]

The left Jacobian can be derived by integrating:

\[ V = \int_0^1 \exp(s\,[\boldsymbol{\omega}]_\times)\,ds \]

Substituting Rodrigues' formula and integrating term by term yields the closed-form expression.

Numerical Implementation

inline Matrix3 Vso3(const Vector3& w)
{
Matrix3 W = skew(w);
double th = w.norm();
if (th < kSmallAngleThreshold)
{
return Matrix3::Identity() + 0.5 * W;
}
double A = std::sin(th) / th;
double B = (1.0 - std::cos(th)) / (th * th);
double C = (1.0 - A) / (th * th);
return Matrix3::Identity() + B * W + C * (W * W);
}

The small-angle check avoids division by near-zero \(\theta^2\).

Inverse (Right Jacobian)

The inverse of the left Jacobian is:

\[ V^{-1}(\boldsymbol{\omega}) = I - \frac{1}{2}[\boldsymbol{\omega}]_\times + \left(\frac{1}{\theta^2} - \frac{1 + \cos\theta}{2\theta\sin\theta}\right) [\boldsymbol{\omega}]_\times^2 \]

This is used when converting from SE(3) to the Lie algebra (logarithm map).
